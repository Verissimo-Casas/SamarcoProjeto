# blinkAlert

### O blink ativa quando quando o gancho mover

    * BlinkSpeed: 1
    * Vai receber um paramentro rangeObject
    * BlinkScale: BlinkSpeed % rangeObject :> 0.35

### O blink ativa quando o operador esta sem EPI

    * BlinkSpeed: 0.35
